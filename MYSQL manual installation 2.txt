================== creating table ===============================


Mysql > show databases; (to check available databases)

Mysql > create database userlogin; (to create new database, database name is "userlogin;")
Query ok, 1 row affected

Mysql > show databases; (to check available databases)

Mysql > use userlogin; (to use created database)
Database changed

Mysql > show tables; (to check tables)
Empty set

Mysql > create table student(name varchar(50),rollnumber int(10), brach varchar(10)); (to create rows & columns in a table using DDL command)
Query ok, 0 rows affected

Mysql > desc student; (to describe created table)
Reply in column table formate

Mysql > insert into student values('Zubair',001,'IT'); (its direct method, data insert into columns in proper formate name, roolnumber, brach) always varchar in '' single quote & inter in normal, after each value "," will be use)
Query ok 1 row affected

Mysql > insert into student(brach, name,rollnumber)values('cs','tom',001); (it is a secure method)

Mysql > SELECT * FROM student; (to check inserted data in the student table)
